# EAMS 资产台账模块 前后端联调核对清单

---

## 一、接口清单

| 功能 | 方法 | URL | 前端函数 | 权限 |
|:----:|:----:|------|----------|------|
| 列表查询 | GET | `/api/asset/list` | `listAssets(query)` | SUPER_ADMIN / ASSET_ADMIN / DEPT_ADMIN |
| 资产详情 | GET | `/api/asset/detail/{id}` | `getAssetDetail(id)` | SUPER_ADMIN / ASSET_ADMIN / DEPT_ADMIN |
| 新增资产 | POST | `/api/asset/add` | `addAsset(data)` | SUPER_ADMIN / ASSET_ADMIN |
| 编辑资产 | PUT | `/api/asset/edit` | `editAsset(data)` | SUPER_ADMIN / ASSET_ADMIN |
| 删除资产 | DELETE | `/api/asset/delete/{id}` | `deleteAsset(id)` | SUPER_ADMIN / ASSET_ADMIN |
| 折旧明细 | GET | `/api/asset/depreciation/{assetId}` | `getDepreciation(assetId)` | SUPER_ADMIN / ASSET_ADMIN |
| 批量导入 | POST | `/api/asset/import` | `importAsset(file)` | SUPER_ADMIN / ASSET_ADMIN |
| 导出Excel | GET | `/api/asset/export` | `exportAsset(params)` | SUPER_ADMIN / ASSET_ADMIN |

## 二、后端接口字段对照

### 列表查询参数 `AssetQueryDTO`

| 字段 | 类型 | 说明 | 前端传参 |
|------|------|------|----------|
| `assetCode` | String | 资产编码精确匹配 | `query.assetCode` |
| `assetName` | String | 资产名称模糊搜索 | `query.assetName` |
| `category` | String | 资产分类字典值 | `query.category` |
| `status` | Integer | 资产状态 | `query.status` |
| `deptId` | Long | 所属部门ID（含子部门） | `query.deptId` |
| `userName` | String | 使用人模糊搜索 | `query.userName` |
| `location` | String | 存放地点模糊搜索 | `query.location` |
| `beginPurchaseDate` | String | 采购日期区间开始 | `query.beginPurchaseDate` |
| `endPurchaseDate` | String | 采购日期区间结束 | `query.endPurchaseDate` |
| `pageNum` | int | 当前页码，默认1 | `query.pageNum` |
| `pageSize` | int | 每页条数，默认10 | `query.pageSize` |

### 列表返回体 `AssetInfo`

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | Long | 资产ID |
| `assetCode` | String | 资产编码 |
| `assetName` | String | 资产名称 |
| `category` | String | 资产分类（字典值） |
| `specification` | String | 规格型号 |
| `status` | Integer | 资产状态 0-5 |
| `deptName` | String | 部门名称（后端填充） |
| `userName` | String | 使用人姓名（后端填充） |
| `location` | String | 存放地点 |
| `purchaseDate` | String | 采购日期 `2026-06-01` |
| `originalValue` | BigDecimal | 原值 |
| `imageUrl` | String | 图片URL |
| `createTime` | String | 创建时间 |

### 新增请求 `AssetAddDTO`

| 字段 | 必填 | 类型 | 说明 |
|------|:----:|------|------|
| `assetName` | ✅ | String | 2-50字符 |
| `category` | ✅ | String | 字典值如 `IT_EQUIPMENT` |
| `specification` | 否 | String | 最大50字符 |
| `snNumber` | 否 | String | 最大50字符 |
| `procurementNo` | 否 | String | 最大50字符 |
| `originalValue` | ✅ | BigDecimal | >0 |
| `purchaseDate` | ✅ | String | `2026-01-15` 格式 |
| `usefulLife` | ✅ | Integer | 1-50 |
| `residualRate` | 否 | BigDecimal | 0-100，默认5 |
| `location` | ✅ | String | 2-50字符 |
| `deptId` | ✅ | Long | 部门ID |
| `userId` | 否 | Long | 使用人ID |
| `status` | 否 | Integer | 默认0（闲置） |
| `imageUrl` | 否 | String | 图片URL |
| `remark` | 否 | String | 最大500字符 |

### 编辑请求 `AssetEditDTO`

| 字段 | 必填 | 说明 |
|------|:----:|------|
| `id` | ✅ | 资产ID |
| `version` | 否 | 乐观锁版本号，防并发覆盖 |
| 其余字段 | 同新增 | 同上 |

### 详情返回 `AssetVO`

| 字段 | 说明 |
|------|------|
| 基本信息 | 同 `AssetInfo` 全部字段 |
| `depreciatedMonths` | 已计提月数 |
| `monthlyAmount` | 月折旧额 |
| `accumulated` | 累计折旧 |
| `netValue` | 当前净值 |

## 三、校验与拦截场景

| 场景 | 前端处理 | 后端处理 | 提示文案 |
|------|----------|----------|----------|
| 资产名称空 | 表单校验 `required` | `@NotBlank` | "资产名称为2-50个字符" |
| 原值≤0 | 表单校验 `min=0.01` | `@DecimalMin("0.01")` | "原值须大于0" |
| 采购日期晚于今天 | 无 | `≤当前日期` | "采购日期不能晚于当前日期" |
| 存放地点空 | 表单校验 `required` | `@NotBlank` | "存放地点为2-50个字符" |
| 在用/借用无使用人 | 前端拦截 | 后端校验 | "状态为【在用/借用】时请指定使用人" |
| 删除在用资产 | 前端拦截+后端拦截 | `status==1`拒绝 | "该资产当前为【在用】状态，请先归还后再删除" |
| 删除借用资产 | 前端拦截+后端拦截 | `status==2`拒绝 | "该资产当前为【借用】状态，请先归还后再删除" |
| 删除盘点中资产 | 前端拦截+后端拦截 | `status==5`拒绝 | "该资产正在盘点中，请等待盘点结束后再操作" |
| 编辑盘点中资产 | — | `status==5`拒绝 | "该资产正在盘点中，请等待盘点结束后再操作" |
| 乐观锁冲突 | — | `version`校验 | "数据已被他人修改，请刷新后重试" |
| 资产编码重复 | — | 唯一索引 | "资产编码已存在"（极少发生） |
